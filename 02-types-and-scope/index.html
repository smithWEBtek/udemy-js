JavaScript Course: Types and Scope

In JavaScript, a variable may store two types of data: primitive and reference.

JavaScript provides six primitive types as:

  • undefined
  • null
  • boolean
  • number
  • string
  • symbol
and a reference type object . ...In other words, the variable that stores a primitive value is accessed by value.
